# Online Payments Fraud Detection System

This project detects fraudulent online transactions using a Machine Learning model integrated with Flask.

## Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
